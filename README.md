# Stationeers mod "GrieferCore" for BepInEx.

# Description:  
Griefer Core is a server-side BepInEx mod for Stationeers focused on implementing a world-based ownership and permission system for player interactions with structures and objects (maybe).
This mod changes the forces created by pressure differences to also take into account the local gravity. On mars, with its weak 3.8m/s gravity, the effect will be almost the same that of Vanilla, but at bodies with stronger gravity, things will be heavier and the wind will have less of an effect on it.

The mod tracks ownership using Steam IDs and is designed to enforce server-side validation of destructive and modifying actions such as building, repairing, upgrading, and deconstruction.

A trusted player system is planned but not yet implemented. The current focus is on building a fully server-sided permission system. Chat-based commands are also planned and intended to be used for managing trusted access in-game in future versions.
